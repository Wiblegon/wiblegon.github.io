ElevenLabs API anahtarı

Ücretsiz hesap: elevenlabs.io → kayıt ol → sağ üst profil → API Keys → anahtar oluştur, buraya yapıştır. Ücretsiz pakette ayda 10.000 kredi var (Flash modelde ≈ 20.000 karakter TTS).

Ses eşleme (ElevenLabs voice ID)

Karakter sesi eklemek için: elevenlabs.io → Voices → Explore → karakteri ara (örn. "Arthur Morgan", "GLaDOS", "villager") → beğendiğini My Voices'a ekle → sesin ID'sini kopyala (ses kartında ⋮ → Copy Voice ID) → buradaki kutusuna yapıştır → kaydet. Boş bırakılan karakterler en yakın klasik sesle konuşur. 4 klasik ses hazır gelir, istersen onları da değiştir.

Elle hak ekle / çıkar (hata düzeltme)

Pozitif sayı ekler, negatif çıkarır. Örn: webhook kaçırdıysa +1, yanlışlıkla verildiyse -1.
